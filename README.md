# Dictionary Booster

Dictionary Booster is a Python desktop application that helps users improve their vocabulary through random word generation, personal word storage, and scheduled desktop notifications.

## Features
- User authentication (login & signup)
- Secure password hashing with salt
- Random word generation using an external API
- Personal dictionary to save words and meanings
- Desktop notifications for vocabulary reminders
- SQLite database for persistence

## Tech Stack
- Python
- Tkinter / CustomTkinter
- SQLite
- Requests
- Threading

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python booster.py
